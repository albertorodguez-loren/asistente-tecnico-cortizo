# asistente-tecnico-cortizo
# 🤖 Asistente Técnico Inteligente - Cortizo Cloud

Este proyecto es un asistente virtual avanzado basado en **Inteligencia Artificial (RAG)** diseñado para resolver dudas técnicas sobre la plataforma Cortizo Cloud, consultando directamente los manuales oficiales de la empresa.

## 🚀 Funcionalidades
* **Consulta de Documentación**: Procesa +20 manuales PDF técnicos en tiempo real.
* **Memoria Contextual**: Mantiene el hilo de la conversación para responder preguntas complejas.
* **Interfaz Web**: Aplicación interactiva desplegada en la nube.

## 🛠️ Tecnologías utilizadas
* **Lenguaje**: Python
* **IA**: Gemini 2.5 Flash (Google AI Studio)
* **Frontend**: Streamlit
* **Procesamiento de Datos**: PyPDF2 para extracción de texto técnico.

## 🔗 Acceso a la App
Puedes probar el asistente en vivo aquí: https://asistente-tecnico-cortizo.streamlit.app/
